# Security Policy

If you discover a security vulnerability in this repository or in any of the apps it covers, please report it privately:

- **Preferred**: [GitHub's private vulnerability reporting](https://github.com/gustavoortega/legal/security/advisories/new)
- **Email**: gortega+pihelm@gmail.com

Please **do not** open public issues for security matters.

I aim to acknowledge reports within 72 hours and provide a fix or remediation plan within 14 days for confirmed issues.

## Scope

- The privacy policies and legal documents hosted in this repo
- The GitHub Pages site at https://gustavoortega.github.io/legal/

For security issues in specific apps, the contact above also applies.
